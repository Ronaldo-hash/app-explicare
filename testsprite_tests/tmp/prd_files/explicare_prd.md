# Explicare - Sistema de Gestão Jurídica

## Descrição
Sistema web para gestão de processos jurídicos com upload de vídeos explicativos e PDFs para clientes.

## Funcionalidades Principais

### Login e Autenticação
- Login com email e senha (admin@admin.com / admin951753)
- Proteção de rotas autenticadas
- Logout do sistema

### Dashboard Administrativo
- Visão geral com estatísticas
- Navegação por menu lateral
- Tema claro/escuro

### Gestão de Clientes
- Lista de clientes agrupados por nome
- Visualização detalhada do cliente
- Campos: email, telefone, endereço, notas
- Edição inline das informações

### Gestão de Processos
- Upload de vídeo explicativo
- Upload de PDF do processo
- Preview do PDF
- Download do PDF
- Cópia de link para compartilhamento
- Analytics de visualizações

### Menu de Navegação
- Visão Geral
- Meus Casos
- Novo Upload
- Configurações
